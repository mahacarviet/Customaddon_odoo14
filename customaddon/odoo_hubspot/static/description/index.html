Automatic Jobs



Hubspot Credentials

  • Once you get Hubspot API Key from your Hubspot Account.
  • Then go to Hubspot>Hubspot Credentials, where you can see Hubspot API and Email.
  • Enter Hubspot API key and email over there.
  • Save the credentials first then click on Test Connection Button. Once you get success to test connection then you can perform other activities.
  • If you face any issue while Test connection then Please verify you API key.


Hubspot Connector

  • To Sync data between Odoo and Hubspot, you have to go to main menu=>Hubspot=>Hubspot Menu.
  • You can see here checkboxes to sync companies, contacts, deals and tickets.
  • Enable respective checkboxes and click Start Execution button.


AUTOMATED ACTION

  • If you want to set auto scheduler to perform sync automatically at a specific time interval, then you can select the auto scheduler option from Hubspot Menu.
  • Once you check the modules to import/export data and select the auto scheduler checkbox and set Execute Every and Interval Unit. Then click on sync data button. In this way, the scheduler will be activated and keep on executing at specific time..


Hubspot Instance Sync

  • If you want to export the record instantly from Odoo to Hubspot when it is created in Odoo, then you have to check the given options in Hubspot credential setting. This will help you to sync following records from Odoo to Hubspot:
    • Create or Delete Company in Hubspot If Created or Deleted in Odoo.
    • Create or Delete Contact in Hubspot If Created or Deleted in Odoo.
    • Create or Delete Deal into Hubspot If Created or Deleted in Odoo.
    • Create or Delete Ticket into Hubspot If Created or Deleted in Odoo.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise (On-premise and Odoo.SH) as well as Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.

You can contact us to request a customization in the app.

Does the Hubspot support Odoo community Version?

Helpdesk does not available in Odoo community version so, Hubspot connector does not support in Odoo Community.
