Connect Odoo with GHN
First of all, GHN Integration App should be available on Odoo system.

Set GHN_Token
Navigate to Sales > Configuration > Settings.
Go to Settings page, under Shipping section, enter GHN_Token.

To get the token:
Login to GHN system https://khachhang.ghn.vn/account. Follow here https://ghn.vn/blogs/thong-tin-tai-khoan/lam-sao-de-dang-ky-tai-khoan to create an account.
On the account page, users can get the token on Token API field. On Your Company page, enter the shop ID on GHN shop_id.

Set Shop_id
Navigate to Inventory > Configuration > Warehouses.
On Your Company page, enter the shop ID on GHN shop_id.

To get shop ID:
Login to https://khachhang.ghn.vn/store
On Quan ly cua hang page, get the shop ID under ID column.

General configuration
Get general address data
Giaohangnhanh supports getting address data on Odoo (such as city, district, ward information).
To get city information
Navigate to Contacts > Configuration > Fed.States.
On Fed. States page, select cities on the list.
After finish, click on Action > Get States.

To get district information
Navigate to Contacts > Configuration > Districts.
On District page, select districts on the list.
After finish, click on Action > Get Districts.

To get ward information
Navigate to Contacts > Configuration > Wards.
On Wards page, select wards on the list.
After finish, click on Action > Get Wards.

Set up address
Admin needs to set up company address and customer address so the system can handle the shipment information when creating orders.

Create shipment method
Admin need to create shipment methods and assign proper service from GHN to the method, which will be used when selecting shipment option for order.
Navigate to Sales > Configuration > Shipping Methods.
On Shipping Methods page, click on Create button.

On the new shipping method page
Enter general information.
Provider: Select GHN Shipping.
GHN Services section appears below, there are three available options: Express, Standard and Bulky And Heavy (this option is available in a certain region).

Use GHN Shipment in orders
Create an order with all necessary information (customer, items, expiration, payment termsâ€¦).
Click Add Shipping to get shipment for the order.

Shipping Method: select a shipping method set before.

Fill necessary information for the package (weight, dimension, payment option, shipment note).

After that, the shipping cost will be updated on Cost field.

Click Update to add shipping cost to the order.
